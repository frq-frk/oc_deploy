Welcome to the Code Compiler API

Use the following endpoints to interact with the API:

POST /api/code/execute

Description: Execute code from an uploaded file.

Parameters:

  • language: The programming language (e.g., "python", "java", "cpp").
  • file: The code file to be uploaded.

Example CURL Command:

curl -X POST -F "language=python" -F "file=@path/to/file.py" http://localhost:8080/api/code/execute

POST /api/code/execute-code

Description: Execute raw base64-encoded code.

Parameters:

  • language: The programming language (e.g., "python", "java", "cpp").
  • code: Base64-encoded string of the code.

Example CURL Command:

curl -X POST -d "language=java" -d "code=$(echo 'System.out.println(\"Hello\");' | base64)" http://localhost:8080/api/code/execute-code
